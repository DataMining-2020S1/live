Big Data Infrastructures

Students are encouraged to download and install Spark and to investigate its machine learning capability.

  • Install / Verify Spark locally
    We will use a Python 3.8 distribution and all additional modules that will be used during this course have been tested to ensure no incompatibilities at this time.

  • Simple Spark/Jupyter examples
    A program to calculate $\pi$, another to count words in text and the opportunity to use Spark's mllib library to classify the iris data.
