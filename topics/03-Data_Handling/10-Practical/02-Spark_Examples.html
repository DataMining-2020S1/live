Spark dataframe examples

Like python, Spark has "borrowed" the dataframe concept from R. Spark dataframes are the preferred data structure for most users, because they support a richer set of operations than either RDDs or Datasets.

Reading and writing data in Spark

The following example shows how Spark can create a dataframe object, change it and display the results of that change.

1
./bin/spark-submit examples/src/main/python/ml/sql_transformer.py

There is also a Jupyter notebook (right click and "Save As" to use) version that takes those basic operations, and shows how data from a Spark context (which "lives" in the HDFS data store that Spark uses, and hence has been distributed across multiple nodes (in this case, the nodes are virtual)) can be read into a pandas (in-memory) dataframe, subjected to more changes, and written back as a Spark dataframe. The notebook has been annotated to explain what is going on.

Transform examples

There are two examples of transformation pipelines.

The first is the classic word count program for Spark, translated to python and stored in a notebook.

The second computes $\pi$ using a geometric approach. This does not use map and reduce as such, but it shows how transformers can be applied in sequence. This example can also be downloaded as a notebook.
